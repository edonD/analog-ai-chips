# analog-ai-chips — Quick Start

## Mission
Research the most advanced analog and mixed-signal AI chips of 2025-2026. Full mission in `program.md`.

## Quick Start
1. Read `program.md`
2. Check `README.md` (what has been found so far)
3. Search. Follow what is interesting.
4. Write `research/<whatever-makes-sense>.md`
5. Update `README.md`, commit, push, repeat

## Key Commands
```bash
git add -A && git commit -m "research: <what you found>" && git push
```

## Rules
- You decide the structure — follow the research, not a predetermined outline
- Verify all numbers, cite sources
- Find the catch in every "analog beats GPU" claim
- NEVER STOP
